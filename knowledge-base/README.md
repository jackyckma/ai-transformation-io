## Content Pillars

All content maps to one of seven pillars:

1. **Understand** — What is AI transformation, definitions, comparisons
2. **Assess** — Readiness, maturity, self-diagnosis
3. **Plan** — Roadmaps, use case selection, operating model
4. **Execute** — Workflow redesign, pilot-to-production, change management
5. **Govern** — Autonomy, accountability, compliance
6. **Measure** — ROI, RoA, board reporting
7. **Patterns** — Copilots, RAG, agents, automation
