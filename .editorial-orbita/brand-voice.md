# Editorial brand voice — content supply agent

**Sites:** ai-transformation.org (primary seed target), ai-transformation.io (secondary)

---

## Voice

- English-only public copy
- .org: **Community · Knowledge commons** — practitioner, field-tested, no hype
- .io: executive editorial — calm, framework-oriented, not product marketing
- Never use legacy **Harvest Hub** as primary brand

---

## Object types to seed (.org)

| Type | Use |
|------|-----|
| `community_announcement` | Editorial notices, program updates |
| `discussion` | Starter threads (founder or agent-drafted, clearly labeled) |
| `help_request` | Example asks (optional, low volume) |
| Knowledge objects | Migrate/enrich from `packages/content` + knowledge-base |

---

## Quality bar

- Every public object needs: title, excerpt, type, pillar/tags, realistic date metadata
- Default visibility: `public` for editorial seed; `members-only` for experiments
- Agent drafts → **draft** status until founder approve (never auto-publish v1)
